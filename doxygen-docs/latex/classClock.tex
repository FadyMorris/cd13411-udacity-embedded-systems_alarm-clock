{\footnotesize\ttfamily void Clock\+::show (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})}



Show the time, alarm, or menu on display. 

This function checks the current state stored in the class member variable {\ttfamily state} and changes the 7-\/segment display accordingly. The blinking is controlled by the {\ttfamily blink\+\_\+state} variable. For example\+: If {\ttfamily blink\+\_\+state = 0b100} (blinking the middle colon), and {\ttfamily display\+\_\+state} = 0b111 (display all the objects hours, minutes, and middle colon), a call to the function every 0.\+5 seconds will cycle the display state betwen {\ttfamily 0b111} to {\ttfamily 0b011}, creating the blinking effect.

The xor operation is used to toggle the display\+: \[ \mathrm{display\_state} = \mathrm{display\_state} \oplus \mathrm{blink\_state} \] \mbox{\Hypertarget{classClock_a00e75ce4c4c79c9a4e94caaaf71e1e30}\label{classClock_a00e75ce4c4c79c9a4e94caaaf71e1e30}}
